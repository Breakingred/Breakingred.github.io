    Tableau Desktop 9.3.8 Mac 中文破解版安装

    1. 下载dmg安装包
    2. 编辑hosts文件：sudo nano /etc/hosts 添加 127.0.0.1 licensing.tableausoftware.com Ctrl+X 确认保存
    3. 打开dmg，安装Tableau.app
    4. 将dmg中的破解文件 libtabui.dylib 替换到Tableau显示包内容里的Contents-Frameworks
    5. 完成安装

    S.M.A.R.T Objectives 用SMART原则衡量项目目标：

    1. Specific
    2. Measurable
    3. Attainable
    4. Relevant
    5. Time-Bound
